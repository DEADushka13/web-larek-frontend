Товар+:
- Название товара                   string
- Категория товара(тег)             string(под стринги можно фильтры и сортировки, 'другое' в них не вписывается, расширяемо)
- Изображение товара                string(можно url, если нет стандартного - regex)
- Цена товара(расширяемо:валюта)    {number?,string}   
- Подробное описание товара         string

+Данные покупателя:
- Способ оплаты                     'online'|'after получения'
- Адрес доставки                    string(можно придумать с адресом что-то)
- Почта                             string(email? regex)
- Телефон                           string(телефон?)

Коллекции:
- +Каталог товаров
- +Корзина с товарами: добавляется total - общая сумма товаров

Действия:
- Данные покупателя:
-- +Выбор способа оплаты и адреса
-- +Указание почта и телефона
-- +Очистка данных

- Каталог товаров:
-- +Заполнить каталог
-- +Получить каталог

- Корзина с товарами:
-- +Добавить товар
-- +Удалить товар
-- +Очистить корзину
-- +Получить список
-- +Получить общую сумму товаров
-- +Отдать общую сумму товаров

Компоненты:
- Карточка товара:
-- Название
-- Описание
-- Цена
-- Категория
-- Изображение
- Форма:
-- Способ оплаты
-- Адрес доставки
-- Почта
-- Телефон

Главная страница               -> Карточка товара -> Модель товаров
Модальное окно товара          -> Карточка товара -> Модель товаров
Модальное окно корзины         -> Карточка товара -> Модель товаров
Модальное окно формы оплаты    -> Форма           -> Данные покупателя
Модальное окно формы контактов -> Форма           -> Данные покупателя
Модальное окно успешной покупки
